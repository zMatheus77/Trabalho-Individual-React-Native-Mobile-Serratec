# Clone da Tela Inicial do X

Projeto individual para a disciplina Desenvolvimento de Aplicações Multiplataforma da Residência em TIC do Serratec. Desenvolvido em React Native com TypeScript, reproduzindo a tela inicial do aplicativo X (Twitter).

## Tecnologias

- React Native
- TypeScript
- Expo

## Componentes Utilizados

- View
- Text
- TextInput
- Image
- StyleSheet

## Estrutura do Projeto

```text
/
├── assets/
│   ├── x-logo.png
│   └── preview.jpg
├── App.tsx
├── styles.ts
├── package.json
└── tsconfig.json
```

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npx expo start
```

3. Escaneie o QR Code com o aplicativo Expo Go ou execute em um emulador.
